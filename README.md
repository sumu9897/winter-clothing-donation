# Winter Donation Web Application

## Live URL
[Winter Donation](https://winter-app-52abe.web.app/)

## Project Overview
The **Winter Donation** web application is a dedicated platform aimed at addressing the challenges faced by vulnerable communities in Bangladesh during the cold season. This platform serves as a bridge between compassionate donors and committed volunteers, enabling a streamlined process for donating essential winter items such as jackets, blankets, and sweaters. 

By fostering collaboration and providing an intuitive user experience, the website empowers individuals to make a meaningful impact, ensuring that warmth and comfort reach those in need during the harsh winter months.

The application features several donation campaigns based on different regions (divisions) in Bangladesh. Users can browse active donation campaigns, view detailed information, and donate items through a simple form.

## Key Features
- **User Authentication**: Users can sign up, log in, and authenticate through Google.
- **Donation Campaigns**: View and contribute to various donation campaigns based on geographic divisions.
- **Donation Form**: Users can submit donation details including item quantity, type, pickup location, and optional notes.
- **Private Routes**: Protected routes for logged-in users, including the donation details page and dashboard.
- **Dashboard**: Displays user profile information with an option to update personal details.
- **Responsive Design**: Fully responsive layout, optimized for mobile, tablet, and desktop.
- **Toast Notifications**: Real-time success and error messages displayed using React Toastify.
- **Winter-Themed Design**: Clean, winter-themed design using Tailwind CSS.
- **Google Authentication**: Users can sign in via Google.
- **Password Toggling**: Password visibility toggle on the registration page.

## Technologies Used
- **React**: JavaScript library for building the user interface.
- **React Router**: For routing and navigation between pages.
- **Tailwind CSS**: A utility-first CSS framework for styling.
- **Firebase Authentication**: Used for user login, registration, and Google sign-in.
- **React Toastify**: For showing toast notifications.
- **Swiper**: For the banner/slider section to display donation campaigns.
- **AOS (Animate On Scroll)**: Animation package for scroll effects.

## Pages and Components
- **Login Page**: Allows users to log in with email/password or Google.
- **Registration Page**: Enables new users to create an account with password validation.
- **Campaigns Page**: Displays a list of all ongoing donation campaigns.
- **Donation Details Page**: Shows detailed information about each campaign and allows users to submit donations.
- **How to Help Page**: Provides instructions on how users can contribute.
- **Dashboard Page**: Displays user profile information and includes an option to update the profile.
- **404 Error Page**: Redirects users to the home page when they visit an invalid route.
